# Plan: Add `--demo` flag for TabGate

## Context
TabGate currently requires real terminal adapters (Terminal.app/Ghostty) to function. A demo mode with fake data lets you test and showcase TUI navigation without needing actual terminal sessions.

## Approach
Add a `--demo` flag to the CLI and a `DemoAdapter` that implements `TerminalAdapter` with pre-populated fake tabs. The demo adapter supports in-memory mutations (close, create, rename) so the full TUI interaction works.

## Changes

### 1. Create `internal/adapter/demo.go` — DemoAdapter
### 2. Modify `cmd/tabgate/main.go` — add `--demo` flag
### 3. Modify `internal/poller/poller.go` — nil-check enricher

## Verification
```bash
go build ./cmd/tabgate && ./tabgate --demo
```
